# SMS Spam Detector with Gradio

This project is a simple spam detection tool that classifies SMS messages as "Spam" or "Not Spam" using Natural Language Processing (NLP) techniques and a Multinomial Naive Bayes classifier. It features a web interface built using Gradio.
